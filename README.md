CPPTools
========

C++ library with miscellaneous classes and helper functions used in some of my
other projects.

In order to build the CPPTools library you will need:

* GNU C++ compiler (>= 4.8)
* CMake (>= 2.8.12)
* gtkmm (>= 3.10.1)
